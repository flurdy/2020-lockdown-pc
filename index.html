Why a new PC

  • Macbook Pro was fine before 2020
    • 2014 MBP 13" i7 16GB
  • Pandemic WFH 2020 cooked it
    • Large FP projects
    • Zoom/Slack with Webcam
    • Screen sharing
    • 4K screen(s)
  • B4: Less FP, small microservices
  • B4: Cloud based kubernetes for dev
  • B4: Test in production
  • B4: Not whole stack locally, Pact integration testing not docker compose
  • Now: CPU over 80 degrees
  • Now: 34" ultrawide broke, "upgraded" to 32" 4k, with another on the way

Specs required

  • Multi-core CPU
  • 32GB or more RAM
  • Fast SSD with capacity
  • Graphics card that handles 4K
  • MacOS preferred, but could be Linux
  • Not noisy
  • Preferable 8 or more cores
  • 32GB ok for now, 64GB preferred long term
  • New NVME disks is lightning fast, esp PCIe 4.0 cards
